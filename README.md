## Analysis Performed

### 1. Descriptive Statistics
- Baseline demographics and echocardiographic parameters summarized using
  `gtsummary` (mean ± SD for normal; median IQR for non-normal; n (%) for categorical)

### 2. Normality Testing
- Graphical: histogram, density plot, Q-Q plot
- Formal: Shapiro–Wilk test

### 3. Correlation Analysis
- Pearson correlation between LVOT VTI and MV CW VTI
- Pearson correlation between DVI-LVOT and DVI-MV

### 4. Bland–Altman Agreement
- Method comparison between DVI-LVOT and DVI-MV
- Bias: −0.05 | 95% Limits of Agreement: −0.36 to +0.27

### 5. ROC Curve Analysis
Two severity definitions were used as reference standards:

- Optimal cut-off selected via **Youden Index**
- AUCs compared using the **DeLong test**
